# Gestor de Tareas Personales

## Descripcion

Este programa es un gestor de tareas personales desarrollado en Python que permite al usuario administrar una lista de tareas pendientes. El sistema incluye funcionalidades para agregar, listar, completar y eliminar tareas a traves de dos interfaces: una consola interactiva y una interfaz grafica.


## Funcionalidades

- **Agregar tareas**: Permite ingresar nuevas tareas con descripcion personalizada
- **Listar tareas**: Muestra todas las tareas con su estado (pendiente o completada)
- **Marcar como completada**: Cambia el estado de una tarea a completada
- **Eliminar tareas**: Remueve tareas de la lista
- **Interfaz amigable**: Menu interactivo en consola o interfaz grafica

## Instalacion de dependencias

1. Instalar las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Instrucciones de uso

1. Ejecutar el programa:
   ```bash
   python task_manager.py
   ```

2. Usar la interfaz grafica para:
   - Escribir una nueva tarea en el campo de texto y hacer clic en "Agregar Tarea"
   - Seleccionar una tarea de la lista y usar los botones para completar o eliminar
   - Ver el estado de las tareas en tiempo real
